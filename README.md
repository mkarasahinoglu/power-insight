# PowerInsight
Web app as a factory management system. View factories, units, energy usages, membership informations, and employee numbers. Create, update, or remove factories and units anytime.
## Technologies
- Node.js
- Nest.js
- Elasticsearch
- MongoDB
- PostgreSQL
- JSON Web Tokens
- Vue.js
- Vuetify
- Axios
- Pinia
## Project is currently live
